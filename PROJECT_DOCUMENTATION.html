üí° App Concept & Purpose

OneYear addresses a fundamental human need: understanding and appreciating the finite nature of time. By visualizing an entire year as a grid of 365 (or 366 for leap years) dots, the app:

  • Creates Time Awareness: Helps users visualize how much of the year has passed and how much remains
  • Encourages Mindfulness: Promotes living in the present by highlighting the current day
  • Motivates Action: Seeing limited time remaining can inspire users to make the most of their days
  • Provides Perspective: Offers a bird's-eye view of the entire year at once
  • Tracks Progress: Shows real-time percentage of year completion

‚ú® Key Features

1. Visual Day Grid

The core feature displays all days of the current year in a responsive grid layout. Each day is represented by a circular dot with:

  • Past Days: Displayed with reduced opacity (60%) in gray (#777)
  • Current Day: Highlighted in vibrant mint green (#00FF99) with pulsing animation
  • Future Days: Shown in gray with lower opacity (25%)
  • Day Numbers: Each dot displays its corresponding day number (1-365/366)

2. Real-Time Progress Tracking

  • Displays current date in real-time
  • Shows percentage of year completed
  • Shows percentage of year remaining
  • Automatically updates at midnight
  • Updates when app returns to foreground

3. Responsive Design

  • Adaptive grid layout that adjusts to different screen sizes
  • Dynamically calculated dot sizes for optimal viewing
  • Supports both portrait and landscape orientations
  • Optimized for various device dimensions

4. Smooth Animations

  • Pulsing animation on the current day using React Native Reanimated
  • Smooth transitions between opacity states
  • Performance-optimized with native animations

5. More Apps Integration

  • Menu button in top-left corner
  • Modal popup showcasing other apps from the developer
  • Direct links to Google Play Store listings
  • Featured apps: Drawing Pad, Flip Flop Game, My Citizen Services
  • About Us section with privacy policy link

6. Leap Year Support

  • Automatically detects leap years
  • Displays 366 dots for leap years, 365 for regular years
  • Accurate day-of-year calculations

Core Components

  • Manages year data state and calculations
  • Handles responsive layout computations
  • Implements auto-update logic (midnight refresh, app resume)
  • Renders the day grid using FlatList for performance
  • Controls modal visibility for "More Apps" section

  • Represents individual days as circular dots
  • Implements pulsing animation for current day
  • Applies conditional styling based on day status
  • Displays day numbers within each dot

  • isLeapYear() - Determines if a year is a leap year
